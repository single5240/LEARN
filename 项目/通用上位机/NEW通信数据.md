```html
┌─────────┬─────────┬─────────┬────────────┬──────────┐
| 帧头     | 类型     | 长度    | 数据        | CRC8    |
| 0xAA55  | (1B)    | (1B)    | (N Bytes)  | (1B)     |
└─────────┴─────────┴─────────┴────────────┴──────────┘
```
### 解释
| 字段   | 长度  | 说明                     |
| ---- | --- | ---------------------- |
| 帧头   | 2字节 | `0xAA55`（固定同步标记，抗干扰性强） |
| 类型   | 1字节 | 区分数据类型                 |
| 长度   | 1字节 | 数据部分的长度（最大支持 256 字节）   |
| 数据   | N字节 | 实际载荷（结构由 **类型** 字段决定）  |
| CRC8 | 2字节 | 校验帧头到数据部分的完整性          |
