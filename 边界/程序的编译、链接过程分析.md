### 编译
#### 预编译指令 pragma
##### 1. **#pragma pack([n])**
```c
#pragma pack(push, 1) // 保存当前对齐，设置新的对齐为1字节
struct MyStruct {
    char a;        // 1 byte
    int b;         // 4 bytes
    char c;        // 1 byte
};
#pragma pack(pop) // 恢复之前的对齐设置

```
##### 2. **#pragma message(“string”)**
- 在复杂的项目开发过程中，`#pragma message`用于在编译时输出自定义的信息或提醒，它可以作为一种在代码中留下注释或提醒的手段。
